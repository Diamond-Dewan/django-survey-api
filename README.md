# Django Survey API

Survey app API using Django Rest Framework

**End Points**

1. `api/v1/qanswer/`    GET/POST
2. `api/v1/question/`   GET/POST
3. `api/v1/question/<id>/`   PUT 

**Auth Ends**

1. `api/v1/auth/login/`
2. `api/v1/auth/logout/`
3. `api/v1/auth/admin-register/`
4. `api/v1/auth/get-auth-token/`
